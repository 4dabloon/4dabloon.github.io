# Интерпретируемый язык программирования hindi Интерпретатор языка написан на языке программирования _F#_ ## Авторы | Имя | Роль в проекте | |:-----------------:|:------------------------------------------------:| | Кузнецов Мстислав | Разработчик парсера | | Личковаха Алексей | Разработчик интерпретатора, создание токенов | | Барабанов Глеб | Создание токенов, обработка ошибок | ## Описание интерпретатора Интерпретация происходит в два этапа: - Сначала парсер `parser.fs` производит преобразование кода в набор токенов используя для этого библиотеку FParsec. - Для всех токенов реализован общий абстрактный класс методом evaluate, который имплементирован для всех токенов. Для хранения переменных использована структура `Map`. Реализации токенов хранятся в файле `tokens.fs`. # Реализованные возможности ## Типизация В языке hindi реализованы следующие типы данных: - `real`: целое число или число с плавающей точкой; - `stri`: строка; - `bool`: логическое значение. ## Операторы В языке hindi поддерживаются следующие операторы: ### Арифметические операторы: - '+': сложение; - '-': вычитание; - '*': умножение; - '/': деление; - '//': целочисленное деление. ### Логические операторы: - '&': логическое И; - '/': логическое ИЛИ. ### Операторы сравнения - '>': больше; - '>=': больше или равно; - '<': меньше; - '<=': меньше или равно; - '==': равно; - '!=': не равно. ### Функции Поддерживается определение и вызов функций. ### Встроенные функции Доступны следующие встроенные функции: - 'hin_real()': Считывает число с плавающей точкой из потока ввода и возвращает его значение; - 'hin_stri()': Считывает строку из потока ввода и возвращает ее значение; - 'hout(expr_1, ..., expr_n): Выводит значения выражений expr_1, ..., expr_n в поток вывода. ### Рекурсивный вызов функций Поддерживается рекурсивный вызов функций. ### Лямбда функции Поддерживается вызов лямбда функций. ### Замыкание Реализована поддержка замыканий. ### Обработка ошибок Осуществляется валидация: - Названий переменных; - Ввода данных. ## Примеры ### Объявление переменных ``` stri d = "Hello world!"; real a = 1; real b = 0.5; stri e = hin_stri(); real c = hin_real(); bool tru = TRUE; bool fls = FLSE; ``` ### Пример объявления функции ``` deff doubler = lmbd (real n) { n * 2 } ``` ### Пример программы реализующей счетчик с замыканием ``` deff f = lmbd (real n) { deff g = lmbd () { hout(n); f(n + 1) } }; deff cnt = f(0); deff cnt = cnt(); deff cnt = cnt(); deff cnt = cnt(); deff cnt = cnt(); deff cnt = cnt(); deff cnt = cnt() ``` ## Пример программы реализующей факториал ``` deff factor = lmbd (real n) { if (n <= 1) { 1 } else { n * factor(n - 1) } }; real n = hin_real(); hout(factor(6), factor(1), factor(n)) real bee = 55555; ```
